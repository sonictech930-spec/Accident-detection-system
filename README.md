# Accident Detection and Alert System 🚨

## Description
This project detects vehicle accidents using MPU6050 and vibration sensor.
It sends SMS with GPS location using GSM module.

## Components Used
- Arduino UNO
- MPU6050
- GSM Module (SIM800/SIM900)
- NEO-6M GPS
- Vibration Sensor

## Features
- Accident detection using acceleration threshold
- Automatic SMS alert
- Google Maps location link

## How It Works
1. MPU detects sudden acceleration.
2. GPS fetches coordinates.
3. GSM sends SMS to emergency contact.
